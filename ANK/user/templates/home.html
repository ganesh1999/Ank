How To Use This Website ?

This website provides analysis tool for the result provided by SPPU. Generally it is very hard to analyse the results for each subject according to pdf format of result provided by SPPU. Considering this problems we have developed an algorithm which analyses the pdf result and directy outputs it, which then also downloadable in pdf format.
